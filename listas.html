Listas de HTML

Listas sin orden

Usa la Etiqueta <ul> como contenedor. Cada elemento de lista ira en una etiqueta <li>

se usa para mostrar listas en las que el orden de los elementos no es importante. Por Ejemplo: La lista de la compra, de asistencia, de materiales escolares.

  • Fruta
  • Pasta
  • Carne
  • Leche

Lista ordenada

Usa la Etiqueta <ol> como contenedor. Cada elemento de lista ira en una etiqueta <li>

Son listas enumeradas. Se usan cuando el orden es importante y los elementos tienen importancia según el orden o son pasos a seguir, por ejemplo.

Receta de cafe con leche

  1. Preparar el cafe
  2. Añadir la leche caliente
  3. Añadir azucar al gusto
  4. Servir

Listas descriptivas

Son listas de termino, cada una de éstos con su propia descripción. Se una la etiqueta <dl>(description list) como contenedor, cada termino irá en un <dt> y cada descripción en un <dd>.

Café
- Bebida de color negro que se bebe caliente
Leche
- Bebida de color blanco que se bebe fria o caliente
