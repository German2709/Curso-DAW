Id's y Clases

Las id's y las clases son atributos que se pueden asignar a los elementos de HTML. Mientras que las clases se pueden asignar a varios elementos, las id son unicas y sólo pueden tenerlas un elemento.

Clases

La principal funcion de las clases es apuntar a varios elementos mediante CSS o la manipulación mediante JavaScript a multiples elementos.

Ejemplo:

Este texto es importante

Este texto es importante

Los elementos tambien pueden tener multiples clases, solo deben estar separadas por un espacio cuando se les asignan el atributo.

Este texto es importante y esta en curisva

Id's

Los ID's apuntan a un unico elemento. No puede haber más de un elemento en la misma id.

Sintaxis: En CSS se declara con #. No puede empezar por un número y no puede contener espacios en blanco
