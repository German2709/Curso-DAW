Fecha y hora en JavaScript

El objeto fecha Date()

Nos permite trabajar con las fechas:

El objeto Date() es estatico. No es un reloj como tal, es un objeto cuyas propiedades se generan cuando el navegador carga el documento.

Creando objetos Fecha

Los objetos de fecha se crean con new Date()

Hay distintas maneras de crear uno. Si no indicamos parametro, el objeto se genera con la fecha y hora actuales.

  • new Date()
  • new Date("string de fecha")
  • new Date(año, mes, dia, horas, minutos, segundos, milisegundos)
  • new Date(milisegundos)

new Date('string de fecha')

Crear un objeto a partir de una fecha concreta. En este caso sera el 7 de febrero del 99. Ref al js en linea 4

new Date(año, mes...)

La fecha se crea introduciendo números. Esto genera ciertas situaciones especiales. ref en js linea 11.

Nota: JavaScript cuenta los meses de 0 a 11. Si se introduce un mes mayor que 11, JS seguira contando los meses del siguiente año.

new Date(milisegundos)

JavaScript almacena las fechas como el número de milisegundos que han transcurrido desde el 1 de enero de 1970.

El tiempo cero(zero time) es el 1 de enero de 1970, 00:00:00 GTM 0.

Ahora el tiempo/fecha es: milisegundos transcurridos desde el tiempo 0. ref al js linea 16.

Imprimir fechas

Escribir new Date() convertirá la fecha automaticamente a un string.

Un formato más facil de leer se obtiene al usar el metodo toDateString() Ref al js linea 24

Tomar datos de fechas (Get Methods)

  • getfullYear() - Retorna el año un número de 4 digitos YYYY.
  • getMounth() - Retorna el mes como número del 0 a 11
  • getDate() - Retorna el dia del mes como número del 1 a 31
  • getDay() - Retorna el dia de la semana como número del 0 a 6
  • getHours() - de 0 a 23
  • getMinute() - de 0 a 59
  • getSeconds() - de 0 a 59
  • getMilliseconds() - de 0 a 999
  • getTime() - Retorna el tiempo o los ms desde el tiempo cero.
