Strings en JS

Un strings es cero o más caracteres escritos dentro de comillas.

Esto lo convierte básicamente en una cadena de caracteres.

Se puede usar tanto comillas simples como dobles. Un string pueden contener comillas siempre y cuando sean distintas de las que rodean el string:

let str = 'Tere "terere" Herrera'

Si imprimimos este string: Tere "terere" Herrera

Metodos de strings

lengths:

Para obtener la longitud de un string se usa el metodo length()

let txt = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
let longitud = txt.length;

slice():

Corta o extrae un parte de un string y la retorna como un nuevo string.

slice() necesita dos parametros. La posición inicial y la final (la final no incluida en el corte).

Ref a linea 7 del JS

substr()

Funciona similar a slice(), pero el segundo parametro indice la longitud del string a recortar. Deprecado

replace()

Reemplaza un parte de un string y lo retorna como un string nuevo.

El primer parametro es el valor a buscar en el string y el segundo es el valor por el que lo reemplazamos.

Estos valores son case sensitive. Es decir, que los valores de mayusculas y minisculas deben coincidir.

Solo se reemplaza la primera coincidencia en caso de haber varias en el string.

Para sortear la diferencia de las mayusculas, se puede usar una expresión regular.

Transformar a minisculas y viceversa

Podemos usar toLowerCase() o toUpperCase() para transformar un string o una parte en minisculas o mayusculas, respectivamente.

Contatenar

Limpiar strings

Podemos usar trim() para limpiar los espacios en blanco redundantes de un string.

Si tenemos un string nombre=" Román" el trim lo dejara como "Román"

charAt()

Retorna el caracter ubicado en el indice especificado como parametro.

Convertir strings en arrays con split()

Si tenemos un string cuyos valores estan separados por comas o por cualquier caracter distinguible, podemos servirnos de esto para transformar en un array.
