Conceptos de JS

Un Programa informatico es una lista de instrucciones que se ejecutan por un ordenador.

En programación estas instrucciones se llamen declaraciones o "Statements".

Un programa de JavaScript es una lista de declaraciones de programación.

Statements o declaraciones

Las declaraciones se componen de:

Valores(varios), operadores(operation), palabras claves(keywords), y comentarios(coments)

Ejemplos: Tenemos una declaración (Statements) que indica al navegador que debe escribir "Hola Mundo" dentro de un elemento HTML con la id="test"

document.getElementById('test').innerHTML='Hola Mundo'

Los Statements, o líneas de código, se ejecutan una a una en orden de lectura.

Punto y coma (semicolor) ;

Se usan para separar los Statements en JS. No son obligatorios pero si muy recomendados. Se usan por buenas practicas de código. Ref. al JS Línea 1.

Espacios en blanco

JavaScript ignora los espacios en blanco, y se usan para hacer el codigo mas legible. Ref al js linea 11

let personas = "Carlos";
let persona="Carlos";

Bloques de codigo

Las declaraciones o Statements se pueden agrupar en bloques, dentro de llaves(curly brackets) {...}

El proposito de esto, es definir una serie de declaraciones para que se ejecuten juntas.

Un ejemplo de declaraciones unidas en bloques son las funciones. Ref al JS linea 16

Palabras clave o keywords

A menudo, muchas lienas de codigo empiezan por una palabra que indica la accion a realizarse.

Estas palabras estan reservadas y no pueden usarse en ningun otro contexto. Algunos ejemplos:

  • var - declara una variable
  • let - delcara una variable de bloque
  • const - declara una constante de bloque
  • if - indica un bloque de codigo a ejecutarse cuando se cumple una condición
  • switch - indica un bloque de codigo a ejecutarse dependiendo de dsitintos factores
  • for - indica un bloque de codigo a ejecutarse en bucle
  • funtion - declara una funcion
  • return - sala de una funcion
  • try - implementa manejo de errores a un bloque de codigo
