Todo sobre let

La palabra clave o keyword let tiene 3 usos principales.

  • Las variables declaradas con let no pueden ser redeclaradas.
  • Las variables definidas con let deben declararse antes de ser usadas.
  • Las variables declaradas con let tienen un alcance (scope) de bloque.

No se pueden redeclarar

Antiguamente en JavaScript, al usar var se puede, accidentalmente, redeclarar y cambiar el valor de una variable.

var x = "Juan";
var x = 0;

No solo perdemos el valor anterior, si no que esto puede generar errores.

let impide redeclarar variables y por tanto es mas seguro.

Alcance de bloque (block scope)

Las variables declaradas con let tienen alcance de bloque, es decir aquellas variables declaradas dentro de un bloque {} no pueden se usadas fuera del bloque. ref js linea 1.

Variables declaradas con let deben ser declaradas antes de usarse

Usando var podemos usar variables mientras estas esten declaradas en cualquier parte del codigo.
