Todo sobre las variables

Formas de delcarar variables

  • Usando var delante
  • Usando let delante
  • Usando const delante
  • Usando nada

Por buenas practicas, usaremos siempre una de las 3 primeras.

var

var es la forma clasica de declarar una variable desde los inicios de Javascript

let y const fueron añadidas al lenguaje en 2015.

var se usa en principiantes como metodo de compatibilidad con navegadores antiguos.

const y let

Por regla general, las variables se declaran con const.

Sin embargo, si el valor de las variables puede cambiar, usaremos let.

Ejemplo con variables fijas o constantes y de valor variable. ref al js linea 1.

El operador de asignación =

En Javascript, el signo de igual = no equivale a "resultado de" como en algebra, sino que es un operador de asignación.

x = x + 5

Esta expresion matematicamente no tiene sentido, pues nunca va a haber un número que equivalga a una suma de si mismo con 5

Sin embargo, en codigo, no solo tiene mucho sentido si no que es una practica muy común. Esta expresion indica que al actuar valor de X le estamos asignando el valor de la suma de x + 5.

Basicamente hemos aumentado el valor de X en 5

Nota en Javascript el operador "igual a" se escribe ==

Tipos de datos

Las variables pueden contener distintos tipos de datos. Pueden ser numeros como 100 o texto como "Juan Eduardo".

En programacion a los valores de texto se les llama cadenas de caracteres o strings

Los strings se escriben dentro de comillas dobles o simples.

si escribes un numero dentro de unas comillas, será tratado como un string de texto. ref de js linea 14.

value = undefined

A menudo en los programas informaticos las variables se declaran sin un valor. El valor pueda que sea algo que sea calculado o que sea provisto por un usuraio, como un imput.

Las variables declaradas sin valor tendran como valor undefined. ref al js linea 23

Redeclarar variables

si se redeclara una variable usando var, la variable mantendrá su valor.

Sin embargo, al hacer esto con let o const el codigo lanzará un error.
