Sintaxis de JavaScript

La sintaxis es la serie de reglas que definen como se construyen los programas. Ref al js linea 1

Valores de JavaScript

hay 2 tipos de valores, fijos y variables.

A los fijos se les llama literales(literals).

A los variables se les llama Variables.

Valores fijos, literales

Los dos tipos de literales son los números y los strings, o cadenas de caracteres.

Los números se escriben con o sin decimales.
10.50
1001

Strings son texto, pueden ir escrito entre comillas dobles o simples.
"Jorge González"
'Jorge González'

Variables

En programación se usan para almacenar valores.

JavaScript se sirve de las keywords var, let, const para declarar variables.

El signo igual se usa para asignar valores a las variables.

Operadores

JavaScript usa operadores aritmeticos(+ - * /) para computar variables.

Expresiones

Las expresiones son una conbinacion de valores, variables y operadores que computan ningun resultado o valor. Ref al js linea 11

Comentarios

Los comentarios son lineas de codigo que son ignoradas. No se ejecutan.

Tienen principalmente dos usos. El primero es explicar el codigo y el segundo es para desactivar cierta parte de nuestro codigo sin necesidad de borrarlo.

En JavaScript hay dos manera de comentar. En linea con // y en bloque /*.....*/

Identificadores/Nombres

los Identificadores se usan para nombrar variables y funciones.

Hay una serie de reglas para nombrar comunes en la mayoria de lenguaje de programación.

Un identificador debe empezar por:

  • Una letra (A-Z o a-z)
  • Un simbolo de dolar "$"
  • O un barra baja "_"

los caracteres siguientes pueden ser numeros, letras, _, $.

No se permite empezar con números.

Uso de mayusculas

Los identificadores son sensiblesa las mayusculas. Esto quiere decir que lastName son dos variables diferentes.

Y por ejemplo, JavaScript no interpreta LET o Let como la palabra clave let

Camel Case

Antes, los programadores usaban guiones - para unir distintas palabras en un identificador. last-name, first-name...

Sin embargo, en JavaScript los guiones están reservadas para la resta. Por lo que los programadores de JS se inclinan a separar las distintas palabras mediante mayusculas, dejando el resto en minisculas.

primerApellido, colorVehiculo, codigoPostal.
