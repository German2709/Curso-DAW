Al ocultar un elemento con display: none la pagina se mostrará como si el elemento no estuviera ahi.

Al ocultar un elemento con visibility: hidden el elemento no se mostrará pero el espacio que ocupa quedará en blanco.

width y max-width

Como hemos mencionado anteriormente, un elemento de bloque siempre ocupa todo el ancho disponible

Asignar una anchura con width eviará que un elemento de bloque se estire a los bordes de su contenedor. Entonces podremos asignar su margen margin: auto; para centrar horizontalmente el elemento respecto a su contenedor. El elemento pasará a ocupar el ancho especificado y el resto del espacio sobrante será repartido equitativamente entre margenes.

La propiedad position

Esta propiedad define el metodo de posicionamiento de un elemento.

Hay 5 tipos de posición.

  • static
  • relative
  • fixed
  • absolute
  • sticky
    •

Los elementos se posicionan con las propiedades de top, left, right, bottom. Sim embargo estas propiedades no funcionarán sin especificar primero la position. Ademas dependiendo del tipo de position las propiedades funcionarán de manera distinta.

position: static

Todos los elementos tienen este valor por defecto.

Las propiedades top, bottom, left y right no afectan a los elementos con posición estatica.

position: relative

Un elemento con este valor se posicionará de manera relativa a su posición normal. Es decir, el centro del eje será sobre si mismo.

Este div tiene position: relative

position: fixed

Un elemento con este valor se posicionará relativamente respecto a la pantalla. Lo que quiere decir que tendra la misma posición incluso scrolleamos la web

position: absolute

Un elemento con este valor se posicionará respecto a su contenedor posicionado( no estatico).

Si un elemento absoluto no tiene ancestros posicionados, usará el body y actuará como un position: fixed.

Este elemento div tiene una position: relative
Este elemento div tiene una position: absolute

position: sticky

Un elemento sticky alernará su comportamiento entre relative y fixed dependiendo de la posición de scroll de la pagina
