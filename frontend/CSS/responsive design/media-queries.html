Media Queries

En CSS3, se introdujo la idea, que en vez de adaptar las webs según el tipo de dispositivo del que se esta visualizando, las adaptaremos según las capacidades.

Esto quiere decir que en vez de revisar si el dispositivo es una pantalla, un movil, una impresora, etc, etc.... Simplemente nos basaremos en los siguientes parametros:

  • Ancho y alto del viewport
  • Ancho y alto de la pantalla del dispositivo
  • Orientacion - Si el dispositivo está en vertical u horizontal
  • Resolución

Usar media queries nos capacita con la posibilidad de hacer paginas hechas a medida para pantallas de escritoriom, portatiles, tablets, moviles...etc.

Sintaxis

una expresion de media queries puede contener una o varias condiciones que se resuelven en true o false 

            @media not|only mediatype and ( condiciones ){
