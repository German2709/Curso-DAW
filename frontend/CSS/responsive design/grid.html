Grid o Rejilla

El sistema grid de CSS permite utiizar una plantilla de rejilla, con filas y columnas, para alinear de manera sencilla paginas web, sin necesidad de usar float o posicionamiento.

Ejemplos grid

Una rejilla consite en un elemento padre con uno o más hijos.

El elemento padre debe tener la propiedad display: grid o inline-grid. Los elementos descendientes se convertiran automaticamente en objetos(items) grid.
