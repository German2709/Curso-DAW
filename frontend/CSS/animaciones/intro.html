Animaciones con CSS

CSS permite animar elementos HTML sin necesidad de recurrir a Javascript.

Que son las animaciones

Una animación es el cambio gradual de un estilo a otro.

Se puede cambiar tantas propiedades de CSS como se quiera, tantas veces como sea necesario.

Para usar animación debe especificarse la regla @keyframes

La regla @keyframes

Esta regla contiene los estilos que el elemento tendrá en determinado momento.

Cuando se especifica una regla los cambios se ejecutarán de manera gradual en ciertos momentos.

Para que la animación funcione, debes asignar un @keyframe al CSS de un elemento.

Propiedades de las animaciones

  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode

animation-delay

especifica el retraso del inicio de una animación

Los valores positivos indican el tiempo que ha de transcurrir para la animación de comienzo. Los valores negativos indican que la animación empezará como si ya hubiese transcurrido el tiempo especificado.

animation-iteration-count

Establece el número de veces que se repetirá una animación.

Se puede establecer un número natural de veces o se puede indicar un número ilimitado con el valor infinite.

animation-direction

especifica si la animación se producirá en orden normal, inverso o irá alternando ciclos. Los valores posibles son:

  • normal La animación se reproduce hacia adelante
  • reverse La animación se reproduce hacia atrás (de 100% a 0%).
  • alternate La animación se reproduce hacia adelante, y luego hacia atrás.
  • alternate-reverse La animación se reproduce hacia atrás, y luego hacia adelante.

animation-timing-function

especifica la curva de velocidad al reproducir una animación.

Puede tener los siguientes valores:

  • ease La animación tendra un inicio lento, luego irá rápido y tendra un final lento. Valor por defecto.
  • linear La animación tendra una velocidad estatica durante toda la duración.
  • ease-in La animación tendra un inicio lento.
  • ease-out La animación tendra un final lento.
  • ease-in-out La animación tendra un inicio y final lento.
  • cubic-bezier(n, n, n, n) Permite al usuario especificar la curva de velocidad mediante una función.
