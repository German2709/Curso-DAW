Selectores de CSS

Los selectores tienen la funcion de encontrar o seleccionar elementos HTML a los que queremos dar estilo

  • Selectores sencillos - Apuntan a elementos basandose en su clase ., id # o nombre h1.
  • Combinadores - Selectores basados en una especifica relacion entre los elementos .contenedor div.
  • Pseudo-clases - Selectores basados en el estado del elemento button:hover.
  • Psuedo-elementos - Selecciona una parte especifica de un elemento.
  • De atributo - selecciona elementos basandose en un atributo o en el valor de un atributo.
