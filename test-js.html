Prueba de JavaScript

El HTML proporciona la estructura del documento, el CSS el estilo visual, entonces es JS(JavaScript) el que proporciona la interacción de la web con el usuario.

Con JS podemos hacer que la pagina reaccione ante las dsitintas acciones del usuario. Hacer clic, pasar el raton sobre un objeto en concreto, hacer scroll, usar el teclado, etc.

Al iguial que el HTML nos provee de la etiqueta style para insertar el CSS. Tenemos la etiqueta scrip para escribir JavaScript. El código tambien puede incorporarse como atributo de los elementos (como sucede con el atributo style) por lo que su uso dependerá del criterio del desarrollador.
